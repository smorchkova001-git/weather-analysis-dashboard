## **Проект №1: Анализ температурных данных и мониторинг текущей температуры через OpenWeatherMap API**

### В ходе работы над проектом было выполнено несколько этапов:

1. **Анализ временных рядов по данным за 2010-2019 гг по 15 городам мира**
- Вычислены скользящее среднее и стандартное отклонение для каждого наблюдения с окном в 30 дней
- Рассчитана статистика по среднему и стандартному отклонению каждого города по сезонам
- Выявлены аномалии в температуре, где аномалия - это отклонение от среднее +- 2 стандартных отклонения

     Этап выполнялся двумя способами с распараллеливанием и без него. В результате было выявлено, что распареллеливание не смогло снизить время выполнения

2. **Мониторинг текущей температуры**
 - Получена текущая температура через OpenWeatherMap API для любого из 15 городов нашего датасета
 - Выявлено, является ли текущая температура аномальной

    Этап выполнялся двумя способами с синхронно и асинхронно. Так как такой анализ имеет смысл только для нескольких городов (для одного нет смысла использовать асинхронность), то расчеты проводились на случай, если подается несколько городов сразу. В результате было выявлено, что асинхронный метод имеет значительное преимущество по скорости выполнения по сравнению с синхронным

3. **Создание приложения на Streamlit**

    Приложение находится по [ссылке](https://weather-analysis-dashboard-smorchkova-juliana.streamlit.app)

    В Streamlit создано три страницы:
    - Анализ исторических данных, на которой для выбранного города выводятся:
        1) Исторические данные по температуре
        2) Основные статистики
        3) Линейный график температуры со скользящим средним и выделением аномалий
        4) Статистика отдельно по сезонам
        5) График с сезонным профилем температур
        6) Гистограмма распределения температуры
    - Кластеризация городов
        1) Реализация KMeans с выделением 4 кластеров
        2) Визуальное отображение результатов кластеризации
        3) Результаты кластеризации в табличном виде
    - Мониторинг текущих погодных условий, на которой для выбранного города выводятся:
        1) Характеристики текущих погодных условий (температура, температура "ощущается как", давление, скорость ветра, влажность, описание)
        2) Находится ли текущая температура в пределах нормы или является аномальной
        3) Статистические данные
        4) Временная метка

4. **Бонусы**
- Отдельный блок с кластеризацией для определения городов со схожими погодными условиями
- Выводится более подробная информация о текущих погодных условиях, а не только температура
- Гистограмма распределения температуры
